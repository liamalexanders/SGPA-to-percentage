# 🎓 What is SGPA?

SGPA (Semester Grade Point Average) measures a student’s academic performance during a single semester.

It is usually calculated using credit hours and grade points earned in subjects.

### SGPA Formula

```text
SGPA = Sum of (Credit Points × Grade Points) / Total Credits
